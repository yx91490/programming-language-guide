# Java 入门指南

## 简介

本指南介绍 Java 语言的基础语法与核心特性，面向零基础或需要系统回顾的读者。

### 本指南内容

- Java 基础语法：变量、类型、字符串
- 关键字与类型系统
- 类与对象、接口与抽象类
- 控制流与函数式编程（Lambda、Stream）
- 集合框架与异常处理
- 并发编程与 Java 17 新特性
