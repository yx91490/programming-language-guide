写给Java程序员的Scala入门指南

简介

本指南面向有Java编程经验的开发者，帮助您快速掌握Scala语言。

本指南的特点

  • ✅ 以Java特性为入口，降低学习曲线
  • ✅ 每个章节包含可执行的代码示例
  • ✅ 重点标注Scala的语法糖特性
  • ✅ 提供特性对比表格，方便查阅
  • ✅ 覆盖Java 17和Scala 2.13的主要实用特性

  3. 关键字 - 关键字列表、关键字对比、Scala特有关键字
  4. 类型系统深入 - 统一类型系统、类型别名、类型边界
  5. 类与对象 - 类定义、构造函数、伴生对象
  6. 接口、抽象类与特质 - 接口、特质、多重继承

  1. 表达式与控制流 - if-else、for推导式、match
  2. 函数式编程 - Lambda、高阶函数、闭包、柯里化

  3. 异常处理 - try-catch、Option/Either/Try
  4. 多线程与并发编程 - Thread、Future、Actor模型、并发工具类
  5. 密封类与枚举 - Sealed Classes、ADT
  6. 隐式与类型类 - 隐式参数、类型类模式
